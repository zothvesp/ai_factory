# Goal Engineering Standards Index

Goal Engineering turns intent into executable, measurable, constrained, and
reviewable work. Every AI-assisted task starts with a goal record.

## Use This Index

Use this page when defining task readiness, decomposing work, setting success
metrics, documenting assumptions, constraints, risks, or exit criteria.

## AI Guidance

- Do not begin substantial work without a goal record.
- Mark assumptions separately from facts.
- Translate vague quality words into constraints, risks, KPIs, or acceptance
  criteria.
- Update Project Brain when a goal reveals durable rules, risks, debt, or
  roadmap changes.
