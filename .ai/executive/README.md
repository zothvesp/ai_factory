# Executive Standards Index

Executive standards define the strategic operating frame for the AI Engineering
OS: mission, vision, authority, stakeholders, constraints, roadmap, success
metrics, release strategy, and shared executive language.

## Use This Index

Use this page when a decision affects direction, prioritization, scope,
governance, stakeholder expectations, release policy, or organizational success
measurement.

## Standards Catalog

| Standard | Use When |
| --- | --- |

## AI Guidance

- Executive standards decide direction; implementation standards decide method.
- Escalate when a task conflicts with mission, constraints, stakeholder
  authority, or release policy.
- Do not invent strategic decisions. Mark assumptions and request authority when
  direction is missing.
- Update Project Brain when executive decisions create rules, risks, roadmap
  changes, or glossary terms.
